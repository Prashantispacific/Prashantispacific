## 〉 About

<br/>

I'm **Prashant** — a systems architect working at the intersection of AI alignment, inference engineering, and aerospace. I build things that are **local-first, deterministic, and sovereign** — no cloud lock-in, no magic boxes.

Currently investigating **catastrophic forgetting** in sub-2B language models through a 4-stage post-training pipeline (SFT → GRPO → DPO → Eval), and designing **PRA-SAT v2.0** — a 1U CubeSat focused on robust, failure-tolerant architecture.

Working from nowhere. Building for everywhere.
