## &thinsp; 〉about

<br/>

I'm **Prashant** — I work at the intersection of AI research and systems thinking. My focus is building AI that is **local-first, deterministic, and sovereign** — no black boxes, no cloud lock-in.

Right now I'm studying how small language models forget what they've learned when trained further, and building tools that let AI run entirely on consumer hardware. On the side, I'm contributing to a 1U CubeSat project called **PRA-SAT v2.0**.
