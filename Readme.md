### 🛠️ Technical Focus & Research
I focus on the intersection of high-performance **inference** (the process of an AI model generating an output from input data) kernels and alignment strategies for **LLMs** and **SLMs**.

* **Current Research:** Investigating **catastrophic forgetting** (the loss of previous knowledge when a model learns new tasks) in the `Indica-1.7B` pipeline using **SFT**, **GRPO**, and **DPO**.
* **Next Milestone:** Developing **param2-instruct** to maximize reasoning capabilities in instruction-following models.
* **Aerospace:** Lead designer for **PRA-SAT v2.0**, a 1U CubeSat focused on robust system architecture.
